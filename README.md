# Nautilus

> [!WARNING]  
> Nautilus is currently in a development beta. Current versions may include security vulnerabilities or broken code, and future updates may contain breaking changes, including database migration incompatibilities.

Nautilus is a solution for FIRST Robotics teams to solve challenges of scouting, attendance, and team management. Nautilus allows teams to customize scouting schematics and dashboards to build the ideal scouting solution, **without having to write a line of code**.
